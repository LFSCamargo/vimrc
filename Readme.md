# That's my vim/neovim configuration

I made this repo to share with you guys my configs for the vim and neovim editor

## Installation

1. Install neovim on your machine `https://github.com/neovim/neovim/wiki/Installing-Neovim`
2. Run the script file to install the package manager and put the config into the right place
3. Open the `nvim` with the following command `nvim +PlugInstall`
4. And that's it your vim has been configured!
